### **Podstawowy dekorator autoryzacji**

```python
# backend-python/app/middleware.py
from functools import wraps
from flask import request, jsonify
import jwt

def require_auth(f):
    """Dekorator wymagajƒÖcy autoryzacji JWT"""
    @wraps(f)
    def decorated_function(*args, **kwargs):
        token = None
        
        # Sprawd≈∫ header Authorization
        if 'Authorization' in request.headers:
            auth_header = request.headers['Authorization']
            try:
                token = auth_header.split(" ")[1]  # Bearer TOKEN
            except IndexError:
                return jsonify({'error': 'Nieprawid≈Çowy format tokenu'}), 401
        
        if not token:
            return jsonify({'error': 'Brak tokenu autoryzacji'}), 401
        
        try:
            # Dekoduj token JWT
            data = jwt.decode(token, Config.SECRET_KEY, algorithms=['HS256'])
            current_user_id = data['user_id']
            
            # Sprawd≈∫ czy u≈ºytkownik istnieje
            from app.models import User
            current_user = User.query.get(current_user_id)
            if not current_user:
                return jsonify({'error': 'U≈ºytkownik nie istnieje'}), 401
                
        except jwt.ExpiredSignatureError:
            return jsonify({'error': 'Token wygas≈Ç'}), 401
        except jwt.InvalidTokenError:
            return jsonify({'error': 'Nieprawid≈Çowy token'}), 401
        
        # Dodaj u≈ºytkownika do kontekstu
        g.current_user = current_user
        return f(*args, **kwargs)
    
    return decorated_function
```

```python
def require_admin(f):
    """Dekorator wymagajƒÖcy uprawnie≈Ñ administratora"""
    @wraps(f)
    def decorated_function(*args, **kwargs):
        # Najpierw sprawd≈∫ autoryzacjƒô
        if not require_auth(f):
            return jsonify({'error': 'Wymagana autoryzacja'}), 401
        
        # Sprawd≈∫ rolƒô administratora
        user_role = get_current_user_role()
        if user_role != 'Admin':
            return jsonify({'error': 'Wymagane uprawnienia administratora'}), 403
        
        return f(*args, **kwargs)
    
    return decorated_function

def get_current_user_role():
    """Pobiera rolƒô aktualnego u≈ºytkownika"""
    from flask import g
    if hasattr(g, 'current_user') and g.current_user:
        if g.current_user.role:
            return g.current_user.role.name
    return None
```

## üîê **BEZPIECZE≈ÉSTWO**

### **Zasady bezpiecze≈Ñstwa:**

1. **Zawsze sprawdzaj autoryzacjƒô** przed sprawdzeniem roli
2. **U≈ºywaj dekorator√≥w** `@require_auth` i `@require_admin`
3. **Waliduj dane wej≈õciowe** przed przetwarzaniem
4. **Loguj pr√≥by nieautoryzowanego dostƒôpu**
5. **U≈ºywaj HTTPS** w produkcji
6. **Regularnie rotuj tokeny JWT**

### **Przyk≈Çad bezpiecznego endpointu:**


## üìù **PODSUMOWANIE**

System uprawnie≈Ñ implementuje **zasada najmniejszych uprawnie≈Ñ** - u≈ºytkownicy majƒÖ dostƒôp tylko do tego, co jest im potrzebne do wykonywania pracy. Administrator ma pe≈Çny dostƒôp do systemu, podczas gdy zwykli u≈ºytkownicy mogƒÖ pracowaƒá tylko ze swoimi danymi.

**Kluczowe elementy:**

- ‚úÖ **3-poziomowa hierarchia r√≥l** (Admin, Manager, User)
- ‚úÖ **Dekoratory autoryzacji** (`@require_auth`, `@require_admin`)
- ‚úÖ **Kontrola dostƒôpu na poziomie endpoint√≥w**
- ‚úÖ **Bezpieczne sprawdzanie uprawnie≈Ñ**
- ‚úÖ **Logowanie pr√≥b nieautoryzowanego dostƒôpu**
